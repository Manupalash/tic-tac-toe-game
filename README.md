  # Tic Tac Toe Game

  A simple Tic Tac Toe game with AI and multiplayer modes, built using HTML, CSS, and JavaScript.

  ## Features
  - Multiplayer mode
  - AI opponent
  - Dark neon theme

  ## How to Play
  1. Open `index.html` in a web browser
  2. Choose between multiplayer and AI mode
  3. If AI mode, select X or O
  4. Play the game!

  ## Technologies Used
  - HTML5
  - CSS3
  - JavaScript (ES6+)
